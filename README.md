# Club Registration

## Description
This Android application allows users to register for clubs, manage memberships, and view club details. It provides a simple and intuitive interface for club administrators and members to handle registrations efficiently.

## Features
- User registration and login
- Join or leave clubs
- View club details and member lists
- Admin panel for club management (optional)
- Input validation for secure registration

## Installation
1. Clone the repository:
   git clone https://github.com/hananadanweye2-debug/ClubRegistration.git
2. Open the project in Android Studio.
3. Build and run on an emulator or Android device.

## Usage
- Users can register accounts and join available clubs.
- Admins can manage club memberships and view member information.

## Technologies Used
- Android Studio
- Java / Kotlin
- SQLite / Firebase
